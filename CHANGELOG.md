# Changelog

All notable changes to `ngx-mermaid-canvas` will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-13

Initial public release.

### Added
- Visual drag-and-drop flowchart canvas (maxGraph) with bidirectional Mermaid sync
- Live Mermaid preview panel
- Subgraph support
- 11 node shapes and 4 edge types, with edge labels
- Select / pan interaction modes with alignment guides
- Auto-layout via dagre
- Undo/redo, copy/paste, keyboard shortcuts
- Shape palette, context menus, and minimap
- Independently configurable text-editor, preview, and palette panels
- Exported `MermaidSerializerService` and `MermaidDeserializerService` for standalone Mermaid conversion
